# Phase 3: 一手源溯源 Prompt

> 仅当 has_material_defects == true 时触发。

---

针对以下物质性缺陷进行一手源溯源验证。

物质性缺陷清单（共 {defect_count} 个）：
{defect_list}

命题背景：
{formalized}

溯源规则：
1. 每个缺陷找 ≥2 个独立来源
2. 来源层级：L1(监管公告/SEC Filing/临床数据/论文) > L2(研报/行业数据库) > L3(自研分析)
3. 推理链不能引用推理链——至少一个实证锚点
4. 总工作量上限：{max_sources} 个来源
5. 每溯源完一个缺陷就输出阶段性结论

对每个缺陷，输出：
- 一手源证据（来源、层级、关键数据点）
- 交叉验证结论
- 概率修正建议（±pp，基于证据而非直觉）
- 修正后该因子的条件概率

如果某缺陷找不到可靠一手源，诚实标注"溯源失败，维持原估计"。
