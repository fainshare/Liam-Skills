# Phase 2 对抗审查：偏差检查清单

> 本清单用于 Phase 2 对抗审查阶段。对每个子命题和整体命题逐项检查。

## 六类认知偏差

### 1. 独立性假设偏差

**检测问题**：子因子之间是否被错误地当作独立事件？

**操作方法**：
- 画因果图：A→B→C 还是 A⊥B⊥C？
- 寻找共同原因（confounders）
- 寻找级联效应（一个成功/失败影响另一个的概率）
- 用条件概率替代：P(B|A成功) vs P(B|A失败)

**典型表现**：
- P(全部成功) = P(A)×P(B)×P(C) 远低于直觉
- 或远高于直觉——两种情况都说明独立性假设有问题

**康方案例**：H3优效→FDA信心↑→Summit股价↑→估值叙事↑。朴素独立相乘得 6.1%，条件概率得 10.9%。

### 2. 时间错配偏差

**检测问题**：分析中使用的数据点是否属于同一时间维度？

**操作方法**：
- 检查每个数据点的时间戳
- 特别注意：用"峰值指标"搭配"当前倍数"，或用"当前指标"搭配"未来倍数"
- 确认命题时间窗口内的假设是否内部一致

**典型表现**：
- "安全边际充足"——但用了峰值收入搭配放量期倍数
- "估值合理"——但参照的是不同发展阶段的可比公司

**康方案例**：用峰值收入(RMB 80亿+)搭配放量期P/S(15-25x)论证安全边际，忽略了峰值时应用成熟期P/S(5-8x)。

### 3. 循环论证偏差

**检测问题**：证据链中是否存在自引用或循环引用？

**操作方法**：
- 追踪每条证据的来源链
- 如果 A 的证据来自 B，B 的证据来自 C，C 的证据又来自 A → 循环
- 特别注意自研报告引用自研报告
- 检查"共识"是否来自同一原始来源的多次传播

**典型表现**：
- "多份报告都支持这个观点"——但它们引用了同一份原始数据
- HV 报告引用另一份 HV 报告作为证据

**康方案例**：IO估值范式报告的 P/S 计算错误被传播到后续分析中。

### 4. 锚定偏差

**检测问题**：初始估计是否过度影响了后续调整？

**操作方法**：
- 记录初始估计值
- 检查最终估计与初始估计的距离
- 尝试"从另一端估计"——如果从 0% 开始调整，会得到同样的结果吗？
- 使用参照类别基准率作为替代锚点

**典型表现**：
- 初始 18%，经过大量修正后变成 16%——只调了 2pp？
- "调整"总是在初始值附近小幅波动

**康方案例**：初始 18% → 一审后 ~16% → 二审后 ~16%。调整幅度小，但这可能是收敛信号而非锚定——需要用参照类别(10-15% base rate)交叉验证。

### 5. 幸存者偏差

**检测问题**：参照案例是否只包含了成功案例？

**操作方法**：
- 列出所有参照案例，标注成功/失败
- 计算成功率基准
- 主动寻找失败案例
- 检查"对标公司"是否有选择性

**典型表现**：
- "百济和信达都实现了三年三倍"——但忽略了同期倒下的几十家 Biotech
- 只用成功者的 P/S 倍数作为参照

**康方案例**：用百济/信达作为可比时，需要注意：(1)两者起点P/S 3-4x 远低于康方 27x；(2)同期大量 Biotech 未实现类似回报。

### 6. 方向偏斜偏差

**检测问题**：本轮审查的所有修正是否偏向同一方向？

**操作方法**：
- 统计每个缺陷的修正方向（↑乐观 / ↓悲观 / →中性）
- 计算偏斜比 = max(乐观数, 悲观数) / 总数
- 阈值：>80% 触发警报

**典型表现**：
- 费曼一审 6 个修正全偏正面 → 用"审查"包装了"确认偏差"
- 每轮修正都"发现之前太保守了" → 可能在合理化乐观

**Lakatos 判据**：
- 进步信号：修正产出了新预测或新发现（如"分布是双峰的"）
- 退化信号：修正只是解释"为什么之前的结论其实是对的"

---

## 使用方式

Phase 2 时逐项检查此清单。对每项：
1. 记录：是否存在该偏差？（是/否/不确定）
2. 如果存在：修正后是否可能跨越 Phase 1 定义的决策边界？（是=物质性 / 否=记录但不强制溯源）
3. 如果物质性：需要什么一手源证据来解决？

## ⚠️ 清单的局限

本清单是方法论中最有操作价值的组件，但它无法检测审查者自身的系统性盲区（自我对抗悖论）。6 类偏差覆盖了常见的认知陷阱，但不是穷尽的——特别是领域特定的偏差（如 Biotech 领域的"管线叙事偏差"）需要额外补充。

对高赌注决策，建议将本清单的审查结果交给独立第三方复核。
