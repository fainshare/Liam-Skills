# 苏格拉底诘问法 — 理论基础

## 核心理念

苏格拉底诘问法（Socratic method / Elenchus）是一种通过系统提问来检验论点可靠性的方法。核心操作不是"证明对方错"，而是"通过追问暴露论点内部的不一致或未经检验的假设"。

### 三问框架

| 层次 | 问题 | 对应 socrates 步骤 |
|------|------|-------------------|
| 第一问 | **你说的对吗？** | 事实核查 |
| 第二问 | **这对所有人都成立吗？** | 对称性检查 |
| 第三问 | **去掉这条，你的论证还能站住吗？** | 独立性检查 |

---

## 第一问：事实核查（Fact Check）

### 检查对象

论点中的**事实声明**（factual claims），即声称描述客观现实的陈述。

### 常见错误类型

| 类型 | 示例 | 检查方法 |
|------|------|---------|
| **数据错误** | "临床试验仅纳入中国患者" → 实际有 38% 非中国患者 | 查原始数据源 |
| **引用失实** | "FDA 拒绝受理" → 实际已受理 | 查官方记录 |
| **选择性引用** | 只引用支持看空的数据，忽略反面数据 | 交叉验证多个来源 |
| **时间错位** | 用 2023 年数据论证 2026 年现状 | 检查数据时效性 |
| **归因错误** | "股价下跌因为业绩差" → 实际是行业系统性下跌 | 对照基准/同行 |

### 核查策略

1. **一手来源优先**: 临床试验查 ClinicalTrials.gov / PubMed，财务数据查 SEC/交易所公告
2. **多源交叉**: 至少 2 个独立来源确认
3. **时效性检查**: 数据时间 vs 论点发布时间 vs 当前时间
4. **量化验证**: 能用数字验证的不用文字描述

## 第二问：对称性检查（Symmetry Check）

### 核心问题

> "同样的批评逻辑，是否也适用于被批评者的对手/基准/对照组？"

这是苏格拉底诘问中最有力的一招——**用对方的逻辑反驳对方**。

### 判断框架

| 对称性结果 | 含义 | 判定 |
|-----------|------|------|
| **不对称** — 批评仅适用于被批评方 | 这是独立弱点，论点有效 | ✅ 独立弱点 |
| **对称** — 批评对双方同等适用 | 这是论证技巧，选择性攻击 | ⚠️ 论证技巧 |
| **反向对称** — 批评对批评方更适用 | 这是投射（projection），论点自反 | 🔄 自反论点 |

### 经典案例

**Brahmer 批评案例**:
- 论点: "HARMONi-6 试验人群（90%男性，排除75+岁）不代表真实患者群体"
- 对称性检查: K 药作为对照组，使用的是**相同入组标准**，因此同样受限
- 判定: ⚠️ 论证技巧 — 如果入组标准是问题，K 药同样有问题，这不构成对依沃西的独立弱点

**成本比较案例**:
- 论点: "公司 A 的毛利率低于行业平均"
- 对称性检查: 行业平均是否包含不同商业模式的公司？如果公司 A 的商业模式本质不同（如垂直整合 vs 纯代工），则行业平均不是合理基准
- 判定: 取决于基准选择是否合理

### 常见对称性陷阱

1. **隐含基准假设**: 批评暗含"应该和 X 一样"，但 X 可能不是合理基准
2. **时间不对称**: 用今天的数据批评昨天的决策（事后诸葛亮）
3. **标准不对称**: 对批评对象用严格标准，对基准用宽松标准

## 第三问：独立性检查（Independence Check）

### 核心问题

> "如果去掉这条论点，整个论证体系还能站住吗？"

### 论证结构类型

**独立论证（强结构）**:
```
论点 A ──→ 结论
论点 B ──→ 结论
论点 C ──→ 结论
```
去掉 A，B 和 C 仍支持结论。每条论点独立贡献。

**链式论证（弱结构）**:
```
论点 A ──→ 论点 B ──→ 论点 C ──→ 结论
```
去掉 A，整个链条断裂。论点之间有依赖关系。

**混合结构（最常见）**:
```
论点 A ──→ 论点 B ──→ 结论
论点 D ──→ 结论
```
去掉 A 会削弱但不摧毁结论；去掉 D 也是。

### 判断方法

1. **依赖识别**: 这条论点是否引用了其他论点的结论？
2. **删除测试**: 假设这条论点被证伪，其他论点是否仍然完整？
3. **连锁反应**: 如果这条论点被证伪，哪些其他论点会同时被削弱？

### 投资场景常见链式论证

```
"临床试验仅中国数据" ──→ "FDA 不会批准" ──→ "全球商业化失败" ──→ "估值应大幅下调"
```

这种链式论证的脆弱性在于：**任何一环被否定，整个链条的说服力都大幅下降**。
- 如果第一步被否定（实际有非中国数据），后续都不成立
- 如果第二步被否定（FDA 已受理 BLA），后续都不成立

**独立论证的例子**:
```
论点 1: 成本上升压缩利润率（独立于临床试验数据）
论点 2: 管理层交接带来不确定性（独立于FDA审批）
论点 3: 地缘政治风险影响海外资产（独立于商业化进度）
```

每条论点独立指向"风险被低估"的结论，去掉任何一条，其他仍然成立。

## 综合评级标准

| 质量评级 | 条件 | 建议 |
|---------|------|------|
| **高** | 事实准确 ✅ + 独立弱点 ✅ + 独立论证 ✅ | 保留，纳入决策参考 |
| **中** | 部分准确 ⚠️ 或 依赖论证 ⚠️ 但核心逻辑成立 | 降级，仅作补充视角 |
| **低** | 事实错误 ❌ 或 论证技巧 ❌ 或 链式脆弱 ❌ | 排除，不纳入决策 |

### 组合判定规则

- 事实核查 = 错误 → 直接 **低**（事实错误是致命的）
- 对称性 = 论证技巧 + 独立性 = 链式脆弱 → **低**
- 事实核查 = 部分准确 + 对称性 = 独立弱点 → **中**
- 全部通过 → **高**

## 认知偏差警觉

苏格拉底诘问的执行者（包括 AI）自身也容易犯以下偏差：

| 偏差 | 表现 | 防御 |
|------|------|------|
| **确认偏差** | 只搜索支持自己看法的证据 | 强制搜索反方观点 |
| **锚定效应** | 被第一个看到的分析锚定 | 多源交叉验证 |
| **可得性偏差** | 高估容易想到的风险 | 系统性检查所有维度 |
| **权威偏差** | 知名分析师说的就是对的 | 检查论证逻辑，不看身份 |
| **叙事偏差** | 好故事 = 好论证 | 分开评估叙事吸引力和逻辑严密性 |
